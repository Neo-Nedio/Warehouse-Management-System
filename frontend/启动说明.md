# 启动说明

## ✅ 项目已创建完成

这是一个完整的Vue 3项目，可以直接使用 `serve` 命令启动。

## 🚀 启动步骤

### 1. 安装依赖

在 `frontend` 目录下运行：

```bash
cd frontend
npm install
```

**如果遇到权限问题**：
- 使用CMD（命令提示符）而不是PowerShell
- 或者先运行：`npm config set cache "%USERPROFILE%\AppData\Roaming\npm-cache"`

### 2. 启动服务器

```bash
npm run serve
```

### 3. 访问应用

浏览器会自动打开，或手动访问：
**http://localhost:3000**

## 📋 项目功能

- ✅ 登录功能（密码登录/验证码登录）
- ✅ 用户管理（增删改查）
- ✅ 商品管理（增删改查）
- ✅ 仓库管理（增删改查）
- ✅ 操作日志查看
- ✅ Token认证
- ✅ 自动API代理（解决跨域）

## ⚙️ 配置

- **前端端口**: 3000
- **后端地址**: http://localhost:8080
- **API代理**: 自动配置，无需手动处理跨域

## 📝 注意事项

1. 确保后端服务在8080端口运行
2. 首次运行需要安装依赖
3. 如果遇到权限问题，使用CMD而不是PowerShell
